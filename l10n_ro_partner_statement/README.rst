Usage
=====

To use this module, you need to be an account adviser and go to:

Accounting -> Reporting -> Romania

Salect the Sale or Purchase Journal and click on the "Print HTML" for
the HTML report or "Print PDF". From the HTML report you can download
the XLS file.
