This module updates the price of a subcontracting purchase adding the
value from the purchase line instead of normal Odoo behaviour of setting
the price in stock. We added this change because you don't know the
price of components that are send out.

Installation
============

To install this module, you need to:

- clone the repository https://github.com/NextERP-Romania/odoo-community
- add the path to this repository in your configuration (addons-path)
- update the module list
- search for "Purchase Subcontracting Price" in your addons
- install the module
